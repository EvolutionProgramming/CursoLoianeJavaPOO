Este repositório foi criado para centralizar e documentar a minha jornada de aprendizado na linguagem Java, acompanhando o renomado curso de Java da **Professora Loiane Groner**. Aqui, registro a evolução dos meus estudos, cobrindo desde os fundamentos da linguagem até os conceitos avançados de **Programação Orientada a Objetos (POO)**.

---

## 📂 Estrutura do Repositório

Para manter o projeto organizado e facilitar a navegação pelo código, a pasta `src` foi dividida seguindo uma lógica clara:

| Padrão de Pasta | Descrição | Objetivo |
| :--- | :--- | :--- |
| `packages` **sem** o nome `exercicios` | Códigos de exemplo e demonstrações teóricas. | Fixar os conceitos explicados pela professora durante as aulas e testar a sintaxe. |
| `packages` **com** o nome `exercicios` | Desafios e laboratórios práticos. | Resoluções de problemas desenvolvidas inteiramente por mim para testar minha lógica. |
